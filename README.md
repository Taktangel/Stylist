# Стилист

AI-помощник, который определяет персональный стиль: **типаж по Кибби**, **цветотип (12 сезонов)**
и **архетип**, и даёт советы по цветовой палитре, силуэтам и аксессуарам. Подключается к чат-боту
(в комплекте Telegram-коннектор).

## Что внутри

| Файл / папка | Назначение |
|---|---|
| `mvp/` | Рабочий MVP: движок анализа, HTTP API, Telegram-бот, база знаний |
| `mvp/kb/` | База знаний: 13 типов Кибби, 12 цветосезонов, 7 архетипов |
| `prototype.html` | Визуальный UX-прототип чат-бота (открыть в браузере) |
| `План_продукта_Стилист.md` | Стратегия, архитектура, команда, roadmap, монетизация |
| `Методология_стилиста.md` | Все методы профессионального стилиста (цвет, фигура, тренды и др.) |
| `ТЗ_MVP.md` | Техническое задание на MVP |

## Быстрый старт

```bash
cd mvp
python engine.py --selftest     # проверка движка и базы знаний
python api.py                   # HTTP API на http://localhost:8000
```

Telegram-бот:

```bash
cd mvp
export TELEGRAM_TOKEN="<токен от @BotFather>"     # Windows: set TELEGRAM_TOKEN=...
export ANTHROPIC_API_KEY="<ключ Claude API>"      # опционально: фото + умный чат
python telegram_bot.py
```

Требуется только **Python 3.9+**, внешних зависимостей нет. Подробности — в [`mvp/README.md`](mvp/README.md).

## Секреты

Секреты читаются из переменных окружения и **не хранятся в репозитории**:

- `TELEGRAM_TOKEN` — обязателен для бота.
- `ANTHROPIC_API_KEY` — опционален; включает анализ фото и умный чат через Claude.

В CI/деплое добавляйте их как секреты (GitHub Actions → Settings → Secrets → `TELEGRAM_TOKEN`,
`ANTHROPIC_API_KEY`).

## Статус

MVP. Опросник + движок работают автономно. Анализ фото (Claude vision) и умный чат-консультант
подключаются ключом `ANTHROPIC_API_KEY` — см. `mvp/README.md`.
